# Spelling Adventure

A simple browser spelling game for young kids. No installs, no build step — just HTML, CSS and JavaScript.

## Play locally

Open `index.html` in a browser (Chrome or Edge recommended for the best speech-voice support).

## How it works

- **Play**: choose **Easy** (scrambled letters + blanks) or **Hard** (spell using the full A–Z alphabet).
- The app says the word aloud, shows a picture, and pronounces each letter as it's tapped.
- Getting the word right shows a celebration with an excited voice line.
- **⚙️ Manage Words** (gear icon on the home screen): add new words (with an optional photo) or remove existing ones. Words are saved in your browser's local storage, so they'll still be there next time you open the app on the same device/browser. Use **Reset to Default Words** to bring back the original list.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repo, go to **Settings → Pages**.
3. Under "Build and deployment", set **Source** to "Deploy from a branch", choose the `main` branch and `/ (root)` folder, then save.
4. GitHub will give you a public URL (usually `https://<username>.github.io/<repo-name>/`) within a minute or two.
